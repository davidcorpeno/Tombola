1) Coloca estos archivos en la misma carpeta: index.html, manifest.json, sw.js y añade 2 iconos (icon-192.png y icon-512.png).
2) Para probar localmente: abre index.html en un navegador moderno. En Chrome/Android puedes usar "Agregar a pantalla de inicio" desde el menú para instalar como app.
3) Para publicar y compartir con amigos: súbelo a Netlify (arrastrar carpeta) o GitHub Pages. Ambas opciones son gratis y rápidas.
